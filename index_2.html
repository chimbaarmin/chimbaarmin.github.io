Skills & Habits

I code using HTML (4 and 5), CSS (2 and 3) and JavaScript (jQuery).

When coding, I like to use Brackets code editor with extensions for Emmet, CSSLint and Autoprefixer.

Front-end architecture I usually use is modified SMACSS and I like to deliver linted and minified code whenever possible.

I also like using Bootstrap, Font Awesome, Google Fonts and other freebies around the Web.

In visual department, I like clear, ordered, gridded designs and usually follow principles of International Typographic Style.
